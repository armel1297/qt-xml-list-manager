# qt-xml-list-manager

Qt Desktop Application zum Speichern von Listeneinträgen in einer XML-Datei.

## Beschreibung

Diese Anwendung wurde mit **C++ und Qt Widgets** entwickelt.  
Sie ermöglicht es dem Benutzer, Einträge zu einer Liste hinzuzufügen und diese in einer **XML-Datei** zu speichern.

Die XML-Datei wird im aktuellen Verzeichnis der Anwendung gespeichert.

## Funktionen

- Hinzufügen neuer Einträge über ein Eingabefeld
- Hinzufügen eines Testeintrags
- Anzeige der Einträge in einer Liste
- Speichern der Einträge in einer XML-Datei
- Benutzeroberfläche erstellt mit **Qt Designer**

## Technologien

- C++
- Qt Widgets
- XML (QXmlStreamWriter)
